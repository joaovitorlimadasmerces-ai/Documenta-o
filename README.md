Nome provisório: Escape Rush
Gênero: Runner / Obstacle Course (Obby)
como o jogo vai funcionar: Nosso jogo é um runner linear, onde o jogador precisa avançar em linha reta superando diversos obstáculos ao longo do percurso.
O principal objetivo é chegar até o final da fase, pulando e desviando dos desafios que aparecem pelo caminho. Para ajudar nessa jornada, o jogador poderá coletar buffs, como energéticos, que aumentam sua velocidade e melhoram seu desempenho temporariamente.
O jogo começa em um lobby, onde o jogador pode se preparar e, ao apertar um botão, entrar na partida.
Durante a corrida, existe uma zona de perigo que se aproxima constantemente por trás. Caso o jogador encoste nela ou seja alcançado, ele morre instantaneamente. Isso cria uma sensação de urgência, exigindo reflexos rápidos e decisões precisas.
